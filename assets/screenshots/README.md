# Screenshots

Add GitHub screenshots here before creating a polished public release.

Suggested files:

- `finder-folder.png`
- `app-in-applications.png`
- `terminal-launch.png`

## Capture Checklist

### 1. finder-folder.png

- Open a clean Finder window in a demo project folder
- Keep the folder name readable
- Show `Codex Now` in the toolbar, context menu, or launch flow if possible
- Avoid showing private filenames, client names, or personal documents

### 2. app-in-applications.png

- Open `~/Applications`
- Show `Codex Now.app`
- Keep nearby apps non-sensitive
- Prefer medium icon size so the app name is readable

### 3. terminal-launch.png

- Launch `Codex Now`
- Wait until Terminal, iTerm, or Warp opens in the selected folder
- Make sure the prompt clearly shows the correct directory
- Avoid displaying API keys, shell history, private repo names, or machine-specific secrets

## Recommended Style

- Use macOS light mode for clarity
- Keep the desktop background simple
- Close unrelated windows
- Crop tightly around the key action
- Export PNG files
