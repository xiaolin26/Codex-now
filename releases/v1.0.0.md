# Codex Now v1.0.0

First public release of `Codex Now`.

## Highlights

- Launch `codex` from the current Finder folder
- Fallback to the last opened folder or `$HOME`
- Auto-detect Terminal, iTerm, and Warp
- Install as a local macOS app in `~/Applications`
- Includes a simple `codex-now` CLI wrapper

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/xiaolin26/Codex-now/main/install.sh | bash
```

## Notes

- macOS only
- Requires the `codex` CLI to already be installed
- No secrets or API keys are bundled in this project
