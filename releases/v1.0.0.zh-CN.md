# Codex Now v1.0.0

`Codex Now` 首个公开版本。

## 这版包含

- 从 Finder 当前目录一键启动 `codex`
- 没有 Finder 当前目录时，回退到上次打开的目录
- 再没有时，回退到 `$HOME`
- 自动识别 Terminal、iTerm、Warp
- 安装后生成本地 `Codex Now.app`
- 同时提供 `codex-now` 命令行入口

## 安装

```bash
curl -fsSL https://raw.githubusercontent.com/xiaolin26/Codex-now/main/install.sh | bash
```

## 说明

- 仅支持 macOS
- 需要本机已安装 `codex` CLI
- 本项目本身不包含任何密钥、token 或账号信息
