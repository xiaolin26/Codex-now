# Codex Now v1.1.0

## What's New

- Auto-detect and auto-install `codex` CLI if it is not found
- If `npm` is available, a dialog prompts the user to install via `npm install -g @openai/codex`
- If `npm` is not found, a dialog guides the user to install Node.js first
- Expanded `codex` search to cover all nvm-managed Node versions (not just v18.20.0)
- Also checks `~/.npm-global/bin/codex` as an additional fallback path

## Behavior

1. Launch `Codex Now`
2. If `codex` is found → open terminal and start `codex` as before
3. If `codex` is not found and `npm` is present → prompt "Install?" → install + launch in one step
4. If `codex` is not found and `npm` is missing → prompt to install Node.js from https://nodejs.org

## Install / Upgrade

```bash
curl -fsSL https://raw.githubusercontent.com/xiaolin26/Codex-now/main/install.sh | bash
```

## Notes

- macOS only
- No secrets or API keys are bundled in this project
