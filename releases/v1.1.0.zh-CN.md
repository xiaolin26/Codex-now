# Codex Now v1.1.0

## 更新内容

- 自动检测 `codex` CLI，未安装时自动引导安装
- 如果本机有 `npm`，弹窗询问是否自动执行 `npm install -g @openai/codex`
- 如果连 `npm` 都没有，弹窗提示先去 https://nodejs.org 安装 Node.js
- 扩展了 `codex` 的搜索范围，覆盖所有 nvm 管理的 Node 版本（不再限于 v18.20.0）
- 新增 `~/.npm-global/bin/codex` 作为额外回退路径

## 使用流程

1. 点击启动 `Codex Now`
2. 已安装 `codex` → 直接打开终端并启动
3. 未安装 `codex` 但有 `npm` → 弹窗询问 → 一步完成安装并启动
4. 未安装 `codex` 且没有 `npm` → 弹窗提示安装 Node.js

## 安装 / 升级

```bash
curl -fsSL https://raw.githubusercontent.com/xiaolin26/Codex-now/main/install.sh | bash
```

## 说明

- 仅支持 macOS
- 本项目本身不包含任何密钥、token 或账号信息
