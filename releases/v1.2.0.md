# Codex Now v1.2.0

## What's New

- Auto-detect login status before launching Codex
- If not logged in, opens browser for OpenAI OAuth login automatically
- Clear terminal prompts guide the user through: login → launch

## Behavior

1. Launch `Codex Now`
2. Already logged in → start `codex` directly
3. Not logged in → show login prompt → open browser for OpenAI OAuth → on success, start `codex`

## Install / Upgrade

```bash
curl -fsSL https://raw.githubusercontent.com/xiaolin26/Codex-now/main/install.sh | bash
```

## Notes

- macOS only
- Uses `codex login` (built into the Codex CLI)
- No secrets or API keys are bundled in this project
