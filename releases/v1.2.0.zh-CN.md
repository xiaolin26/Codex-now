# Codex Now v1.2.0

## 更新内容

- 启动前自动检测登录状态
- 未登录时自动打开浏览器完成 OpenAI OAuth 授权
- 终端里有清晰的提示引导用户：登录 → 启动

## 使用流程

1. 点击启动 `Codex Now`
2. 已登录 → 直接启动 codex
3. 未登录 → 弹出登录提示 → 跳转浏览器完成 OpenAI 授权 → 授权成功后自动启动 codex

## 安装 / 升级

```bash
curl -fsSL https://raw.githubusercontent.com/xiaolin26/Codex-now/main/install.sh | bash
```

## 说明

- 仅支持 macOS
- 使用 codex CLI 内置的 `codex login` 命令
- 本项目本身不包含任何密钥、token 或账号信息
