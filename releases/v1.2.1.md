# Codex Now v1.2.1

## Bug Fixes

- Fix: app unresponsive on second launch due to temp file collision
- `mktemp` on macOS does not expand `XXXXXX` suffixes in the `/tmp/file-XXXXXX` format, causing a "File exists" error on every launch after the first
- Switched to `mktemp -t` which correctly generates unique temporary file names on macOS

## Install / Upgrade

```bash
curl -fsSL https://raw.githubusercontent.com/xiaolin26/Codex-now/main/install.sh | bash
```
