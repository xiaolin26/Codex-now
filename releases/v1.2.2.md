# Codex Now v1.2.2

## Bug Fixes

- Fix: app can now be launched multiple times without becoming unresponsive
- Root cause: launcher was writing temp script files and relying on them, which caused macOS Launch Services to block re-launches
- Fix: removed all temp script files; commands are now passed directly to Terminal as inline strings, same approach as the reference implementation

## Install / Upgrade

```bash
curl -fsSL https://raw.githubusercontent.com/xiaolin26/Codex-now/main/install.sh | bash
```
